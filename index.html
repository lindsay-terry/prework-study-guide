HTML

  • The body element represents the visibile content shown to the user.
  • Use semantic elements whenever possible.
  • Element refers to entire element, from opening to closing tag.
  • Tag only refers to what appears inside angle brackets.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Don't make changes in the main branch! Check what branch you are working in before making changes.
  • git add -A: Add the changed files in our working directory to the staging area of Git.
  • git commit -m "descriptive comment": Commit staged changes to repository with a description of what was done.
  • git pull origin main: Verify main branch is up to date before opening a pull request.
  • git push origin branchname: Push up all changes made to working branch.
